# Demo artifacts

This folder contains representative sample outputs from `trailtraining`.

## Files

- `rollups/combined_rollups.json`
  Example merged training + recovery rollup used as forecasting context.

- `plans/training-plan.json`
  Structured generated weekly training plan.

- `plans/training-plan.txt`
  Human-readable rendered version of the weekly plan.

- `status/recovery-status.md`
  Example recovery summary output.

- `status/meal-plan.md`
  Example meal-planning output.
